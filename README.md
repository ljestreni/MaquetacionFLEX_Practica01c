# MaquetacionFLEX_Practica01c

Proyecto de maquetación web responsive utilizando CSS Flexbox.

## Descripción

Esta práctica consiste en la maquetación de una página web inspirada en el videojuego "Arc Raiders", utilizando Flexbox para distribuir los elementos de la interfaz de manera flexible y adaptativa.

## Contenido

El proyecto incluye:

- Un encabezado con títulos descriptivos.
- Un menú lateral de navegación.
- Una sección principal dividida en filas con tarjetas informativas.
- Uso de imágenes responsivas.
- Un pie de página con información de contacto e iconos de redes sociales.

## Desarrollo

El desarrollo se ha centrado en el uso de la propiedad `display: flex` para organizar el layout principal y los componentes internos. Se han aplicado estilos CSS para manejar la alineación, el espaciado y la estética general (colores oscuros, bordes sutiles).
